Instructions:


Firstly, click the clone button and copy the link that drops down.
Next, use your command terminal to cd into a folder that you want to drop the repository in.
Then, do git clone (link)
Now, you should have the repository. Follow these directions below

use the terminal/command prompt to cd into the folder "CaseStudy"
make sure nodeJS is installed
1. run the command "npm install"
2. run the command "npm start"
3. go to your browser and type in "localhost:3000"
4. the website should load. Warning its very barebones.
